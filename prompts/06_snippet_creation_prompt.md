# Snippet Creation Prompt

```text
Create 4-5 short public-facing teaser snippets from the final video.

Requirements:
- choose the most visually impressive moments
- avoid private logos, names, and sensitive screenshots
- if needed, crop or mask only the sensitive region
- preserve the premium look and feel
- also create a README-safe preview GIF or poster frame

For each snippet, specify:
- time range
- why it was chosen
- whether any masking/cropping is needed
- recommended filename
```
