# Visual Direction Prompt

```text
Use the script as a video director, not just as a summarizer.

For every narration segment, specify:
- what should be on screen
- which asset to use
- what text appears
- what is the focal point
- where to zoom or pan
- whether to use a real screenshot or a designed insert
- what motion style to borrow from reference animations
- what emotional beat the scene should create

Be specific:
- exact screenshot region to highlight
- exact labels or callouts
- exact type of animation
- exact moment where the solution reveal happens

The final output should read like a scene-by-scene direction strategy for a premium SaaS/product-launch video.
```
