# Voiceover And Caption Mapping

## Mapping Rules

- captions should support the narration, not duplicate giant center-screen text
- center-title scenes may suppress bottom captions
- KPI text should appear exactly on the metric phrase
- fades should be gentle

## Example Map

| VO segment | Caption treatment | Screen treatment |
| --- | --- | --- |
| Title + tagline | no bottom captions if centered on screen | title reveal |
| Scale line | full caption | screenshot + count reveal |
| Decision questions | segmented captions | constellation + list build |
| Existing workflow line | full caption | workflow baseline screen |
| AI layer reveal | segmented captions | highlight columns + modal |
| Engine description | grouped captions | process or agent flow |
| Proof scenes | short segmented captions | artifact + result |
| KPI lines | full caption until final centered lockup | dashboard/KPI screen |
| Final lockup | no bottom captions if the exact line is centered | end title |
