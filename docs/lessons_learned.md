# Lessons Learned

## 1. Context Beats Style At The Start

The strongest motion direction came only after the business workflow was understood properly. Visual polish without process understanding feels generic very quickly.

## 2. Premium Does Not Mean Busy

The best scenes usually had:

- one strong focal action
- one proof asset
- one clear caption or headline

Too many simultaneous ideas reduced confidence.

## 3. Real Screenshots Create Trust

Even when the story is backend-heavy, a few well-chosen screenshots anchor the audience. Use them for credibility and let designed inserts explain the invisible logic around them.

## 4. References Should Drive Choreography, Not Copying

Template websites are most useful when they teach:

- how the camera moves
- how elements layer
- how reveals are paced
- how emphasis shifts

Rebuild those behaviors in your own system.

## 5. Voiceover Is A Timing Instrument

If a metric appears too early or too late, the scene feels weaker even if the animation is good. Spoken emphasis should guide motion emphasis.

## 6. Captions Need Design Care

Captions that hard-cut in and out can cheapen a premium video. A subtle fade and a dedicated caption band help a lot.

## 7. One Scene Pattern Can Save Many Future Projects

Reusable patterns such as:

- title reveal
- screenshot pan
- modal focus
- process steps
- KPI settle

become a compounding asset library for future launches.

## 8. Public Documentation Needs Its Own Pass

Turning a private production into a portfolio-ready repo takes separate work:

- anonymization
- prompt cleanup
- repo writing
- snippet selection
- preview image creation

Plan for that explicitly.
